Fitness Quest

Fitness Quest is a fitness application for Android devices. Our project is to create an fitness application that gamifies the daily fitness experience. The gamification theme is set around a warrior on an adventure. Each user can create quests for him/herself every day which consist of real exercises or can complete existing recommended exercises.
By Sonny Tu, Syed Wasi Haider, Mark Liao and Raymond Chow
